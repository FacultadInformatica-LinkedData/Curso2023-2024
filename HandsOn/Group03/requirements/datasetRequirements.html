Dataset Requirements

Satisfacemos todos lo requisitos obligatorios de forma que:

R1. Los datos se encuentran en uno de los dominios de ciudad inteligente seleccionados: Utilizamos los datos de Basilea
R2. Los datos están disponibles como un archivo CSv: Sí, lo están
R3. Los datos tienen una licencia abierta para que puedan ser publicados: Tienen licencia Licencia CC BY 3.0 CH por lo que cumple la condición
R4. Los datos se pueden vincular fácilmente con entidades genéricas del mundo real (por ejemplo, ubicaciones): Se nos proporcionan coordenadas por lo que se puede vincular fácilmente.

Requisitos opcionales:

R5. Existe documentación para los datos (opcional): Sí se cumple
R6. Los datos pueden provenir de múltiples fuentes de datos (opcional): No se cumple

Anotaciones importantes: En nuestro dataset no se especifica el nombre de la "smart city" pero sí se proporcionan las coordenadas por lo que hemos podido verificar que se trata de la ciudad suiza Basilea. Además, realizando esta comprobación nos hemos dado cuenta que las coordenadas de algunos datos son erróneas por la pérdida del primer número, aun así creemos que podemos realizar el trabajo sin inconvenientes puesto que se puede arreglar manualmente.
