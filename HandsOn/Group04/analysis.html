Dataset analysis

Variable Type Range Description Problemas
Año Integer [2020/2023] Valor que identifica el año al que pertenece la medición
Mes Integer [1/12] Valor que identifica el mes al que pertenece la medición Altamente dependiente del año y vinculado a ella, juntar las dos columnas del dataset puede que sea mejor idea que tratarlas independientemente.
TipoEdificio String Not Unique Indica la actividad para la que estaba intencionada el edificio. Está dividido en 5 tipos de edificio
Edificio String Not Unique Indica el nombre del edificio.
Dirección String Not Unique Alberga los atributos calle, número, código postal y ciudad. Puede incluir información extra o en algunos casos incompleta.
Distrito String Not Unique Indica el distrito de Madrid al que pertenece el distrito
Código Distrito Integer [1/21] Número identificador del distrito
Barrio String [11/215] Indica el barrio de Madrid al que pertenece el edificio
Código Barrio Integer Not Unique Número identificador del Barrio Las decenas son el número del distrito, la unidad el identificador del barrio.
Coordenada-X Integer [433337,1/451675] Indica la Coordenada X en la que se localiza el edificio
Coordenada-Y Integer [4465173,9/4484220,7] Indica la Coordenada Y en la que se localiza el edificio
Longitud Double [-3.76468/-3.73835] Longitud de su ubicación Los valores depende de la herramienta que se utilice pueden estar corruptos.
Latitud Double [40.426646/40.479004] Latitud de su ubicación Los valores depende de la herramienta que se utilice pueden estar corruptos.
Sensor String Not Unique Sensor empleado para leer la medición de energía. Es un dato que para nuestra base de datos resulta irrelevante.
Tipo String Not unique Indica si el tipo es Externo, Procesado o telemedido
Clase String Not Unique Indica el tipo de energía que se utiliza, la mayoría de edificios utilizan más de una.
Unidades String Not Unique Define el tipo de unidad que se utiliza. Parece ser consistente con la clase de energía.
Consumo String Not Unique Define la cantidad de energia gastada por cada edificio. Hay edificios que no tienen datos o con información incompleta, también hay edificios con energía negativa.
