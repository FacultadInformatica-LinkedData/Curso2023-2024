Used Datasets

count.csv

Editor: City Hall of Paris, Link: https://www.paris.fr/

Owner: City Hall of Paris

License: Open Data Commons Open Database License (ODbL)

Description: This dataset contains the following columns:

  • id_compteur: Unique identifier for the counter
  • nom_compteur: Name of the counter (Address where it is with the direction)
  • name: Name of the counter (Address without direction)
  • sum_counts: Sum of counts
  • date: Date of when it was counted
  • installation_date: Installation date of the counter
  • latitude: Latitude coordinates
  • longitude: Longitude coordinates
  • counter: Counter code
  • street: The overall name of the street

velib-emplacement-des-stations.csv

Editor: City Hall of Paris, Link: https://www.paris.fr/

Owner: City Hall of Paris

License: Open Data Commons Open Database License (ODbL)

Description: This dataset contains the following columns:

  • stationcode: Unique station code
  • name: Name of the station
  • capacity: Capacity of the station, how many bicycles can be max there
  • coordonnees_geo: Geographic coordinates, Long - Latitude

comptage-velo-compteurs.csv

Editor: City Hall of Paris, Link: https://www.paris.fr/

Owner: City Hall of Paris

License: Open Data Commons Open Database License (ODbL)

Description: This dataset contains the following columns:

  • id_compteur: Unique identifier for the counter. Id combined with channel_id
  • nom_compteur: Name of the counter (Address where it is with the direction)
  • id: Identifier which is used for id_compteur - not unique
  • name: Name of the counter (Address without direction)
  • channel_id: Channel ID
  • channel_name: Direction (North, West, etc.)
  • installation_date: Installation date. Year, month and day
  • url_photos_n1: URL for photos of the counter -> JPG files
  • counter: Counter specific name
  • coordinates: Coordinates of the counter
  • test_lien_vers_photos_du_site_de_comptage_: Test link to counting site photos
  • id_photo_1: https or http protocol
  • url_sites: URL for sites
  • type_dimage: Image type: jpg or png etc
